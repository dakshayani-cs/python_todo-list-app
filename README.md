# Python To-Do List Application

## Project Description
This is a menu-driven To-Do List application developed using Python.
It allows users to add, view, and delete tasks using a simple interface.

## Features
- Add Tasks
- View Tasks
- Delete Tasks
- Exit Program

## Technologies Used
- Python
- Basic Programming Concepts (Lists, Loops, Conditions)

## Purpose
This project was created as a beginner-level Python project for learning programming and building basic logic.
